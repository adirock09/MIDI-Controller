I started with layout It has 40 Keys in 10 x 4 Layout.Top two layers are for High octave and bottom two for Low octave the three sliders are for Filter,Reverb,Modulation .I used Matrix so it uses less Gpios, It is based on Raspberry Pico.
PCB made in Kicad and modeling done in Fusion 360. Firmware is written in circuit python and uses Adafruit MIDI Library.
